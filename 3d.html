3D Animation PipelinePersonal Project


2023
Role - 3D Generalist
I studied this 3D animation production process to expand my interest in animation content and adapt its capabilities to current developments in the digital content industry.
Taking the theme of children's games, the production flow includes several important processes in 3D animation production.
Modeling
In keeping with the children's game theme, this project focuses on two objects. The first object is a character design of a child wearing casual clothes with a hat and shoes, and the second object is a swing game model. The reason for choosing these two objects is to learn how to make an organic 3D model (a human character object) and a hard surface object (a swing object).
Material & Texturing
In giving material and texture to these two objects, using the shading process and UV map in Blender to adjust the color and texture of the character object and swing object.
Rigging
The process of rigging the character object is done with the help of the Basic Human (Meta-Rig) feature in Blender, and the swing object is done using manual method by grouping the parts of the swing that I want to animate with an empty plain axes object in Blender. This is also another reason for choosing these two objects to practice rigging human characters and hard surface objects.
Compositing
This process has several stages. The first stage is to set the layout for the placement of the two objects and the camera's point of view. The second stage is to adjust the lighting so that the colors and shadows of the objects displayed can match the mood of the scene composition I want to display.
Animate & Render
In this project, the result of combining the animation of a character object and a swing object is a looping animation of a child character playing on a swing, and the result of the basic animation of the natural movement of a character is a looping animation of a child character's walking cycle. Both of these animations are rendered using the Cycles Render Engine in Blender.
Other works
Some of these other works were created to deepen my understanding of 3D animation by studying the basics and techniques that are useful in the industry.

3D Animation PipelinePersonal Project


2023
Role - 3D Generalist
Saya mempelajari proses produksi animasi 3D ini untuk memperluas minat saya pada konten animasi dan untuk menyesuaikan kemampuan dengan perkembangan industri konten digital saat ini.
Mengambil tema permainan anak, alur pembuatannya sudah memuat beberapa proses penting dalam produksi animasi 3D.
Modeling
Sesuai dengan tema permainan anak, project ini berfokus pada dua objek. Objek pertama berupa desain karakter seorang anak memakai pakaian kasual dengan properti topi dan sepatu, lalu objek kedua berupa model permainan ayunan. Alasan dipilihnya kedua objek ini adalah untuk mempelajari pembuatan 3D model organic (objek karakter manusia) dan hard surface (objek ayunan).
Material & Texturing
Dalam pemberian material dan texture pada kedua objek ini menggunakan proses shading dan UV map di Blender untuk mengatur pemberian warna dan tekstur pada objek karakter dan objek ayunan.
Rigging
Dalam proses rigging pada objek karakter menggunakan bantuan dari fitur Basic Human (Meta-Rig) di Blender dan untuk objek ayunan menggunakan cara manual dengan cara grouping bagian ayunan yang ingin dianimasikan dengan empty plain axes object di Blender. Hal ini juga merupakan alasan lain dipilihnya kedua objek ini, yaitu untuk mempelajari cara rigging objek karakter manusia dan rigging objek benda mati.
Compositing
Dalam proses ini ada beberapa tahap, tahap yang pertama mengatur layout penempatan kedua objek dan mengatur sudut pandang kamera, lalu tahap yang kedua mengatur pencahayaan agar warna dan bayangan objek yang ditampilkan bisa sesuai dengan suasana komposisi scene yang ingin ditampilkan.
Animate & Render
Dalam project ini dilakukan pembuatan animasi gabungan dari objek karakter dan objek benda mati dengan hasil berupa animasi looping karakter anak yang sedang bermain ayunan, serta dilakukan pembuatan animasi dasar gerakan alami sebuah karakter dengan hasil berupa animasi looping siklus berjalan karakter anak. Kedua hasil animasi ini di-render menggunkan Cycles Render Engine di Blender.
Karya lainnya
Beberapa karya lainnya ini dibuat untuk memperdalam pemahaman pembuatan animasi 3D dengan mempelajari dasar dan teknik-teknik yang berguna untuk diterapkan di industri.
